# Security Policy

## Reporting a Vulnerability

Please report security vulnerabilities through GitHub's **private vulnerability reporting**.

1. Go to the [Security tab](https://github.com/nvandessel/floop-bench/security) of this repository
2. Click **"Report a vulnerability"**
3. Fill in the details

**Please do not open public issues for security vulnerabilities.**

## Response Timeline

- **Acknowledgment**: Within 48 hours
- **Initial assessment**: Within 1 week
- **Fix or mitigation**: Dependent on severity, targeting 30 days for critical issues

## Scope

This policy covers the floop-bench harness code and configuration. The benchmark relies on external tools (floop CLI, mini-SWE-agent, SWE-bench Docker images) which are governed by their own security policies.
